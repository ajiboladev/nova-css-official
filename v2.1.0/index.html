Getting Started

Add Nova CSS to your project quickly and easily

CDN Installation

Add this link to your HTML head section:

<link rel="stylesheet" href="https://ajiboladev.github.io/nova-css/v2.1.0/css/style.css">

NPM Installation

Install via NPM:

npm install nova-css
