Outline

Enhance element focus and style with Nova CSS outline utilities

⚠️Note:Outline utilities are currently experimental and may not work as expected in this version. Full functionality will be fixed and improved in the next release.

Outline Utilities

Define focus and element boundaries with consistent outline styles

Outline Width Utilities

Class Property Value
.nv-o-0 outline 0
.nv-o-1 outline 1px
.nv-o-2 outline 2px
.nv-o-3 outline 3px
.nv-o-4 outline 4px
.nv-o-5 outline 5px

Example

This box has outline
<div class="m-4 p-4 bg-primary text-white rounded nv-o-5 nv-o-primary nv-o-dashed">Content</div>

Outline Style Utilities

Class Property Style
.nv-o-solid outline solid straight line
.nv-o-dashed outline dashed dash line
.nv-o-dotted outline dotted dotted line
.nv-o-double double outline straight line x2

Example

This box has padding top
<div class="nv-pt-4 nv-bg-danger nv-text-light">Content</div>
