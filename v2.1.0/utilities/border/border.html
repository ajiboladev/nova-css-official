Design clean and structured layouts with Nova CSS border classes

Note: all Nova Css Documentation are started with .nv-

Border Utilities

Style and customize element borders easily using Nova CSS border utilities.

Border Utilities

Class Property Value
.nv-border-0 border width 0
.nv-border-1 border width 1px
.nv-border-2 border width 2px
.nv-border-3 border width 3px
.nv-border-4 border width 4px

Example

This box has border-4
<div class="nv-p-4 nv-border-4 nv-text-white">Content</div>

Border Style Utilities

Class Property
.nv-border-solid Straight line
.nv-border-dashed dashed line
.nv-border-dotted dotted line
.nv-border-none none

Example

This box has dashed border
<div class="nv-p-5 nv-bg-danger nv-text-light nv-border-dashed nv-border-4">Content</div>

Border Side Utilities

Class Property Value
(.nv-border-top-1)- (.nv-border-top-4) border top 1 - 4 1px - 4px
(.nv-border-right-1)- (.nv-border-right-4) border right 1 - 4 1px - 4px
(.nv-border-bottom-1)- (.nv-border-bottom-4) border bottom 1 - 4 1px - 4px
(.nv-border-left-1)- (.nv-border-left-4) border left 1 - 4 1px - 4px

Example

This box has border left-4
<div class="nv-pb-5 nv-bg-light-dark nv-text-light nv-border-left-4 nv-border-secondary nv-border-dashed">Content</div>

Border Color Utilities

Class Property Color
.nv-border-primary border primary
.nv-border-secondary border secondary
.nv-border-danger border danger
.nv-border-warning border warning
.nv-border-light border light
.nv-border-light-dark border grey

Example

This box has danger border
<div class="nv-p-5 nv-text-dark nv-border-danger nv-border-2">Content</div>

Border Radius Utilities

Class Property Value
.nv-radius-0 border radius 0
.nv-radius-sm small radius 0.25rem
.nv-radius-md medium radius 0.5rem
.nv-radius-lg large radius 1rem
.nv-radius-full full radius 50%

Example

This box has full radius
<div class="nv-p-5 nv-bg-warning nv-text-danger nv-text-center nv-radius-full">Content</div>
