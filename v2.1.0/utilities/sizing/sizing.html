Nova CSS Responsive Engine (Sizing)

Nova CSS Responsive Engine for sizing lets you control width, height, and overall layout behavior across different screen sizes with simple, intuitive classes. Using breakpoint prefixes likexs:, sm:, md:, lg:, and xl:, you can adjust element dimensions responsively without writing extra media queries. Whether it’s full-width containers, fixed cards, or adaptive images, Nova’s sizing system ensures your design stays consistent and flexible across all devices.

Responsive Engine

breaking point Property Value
xs: extra small ≥0px
sm: small screen ≥576px
md: medium screen ≥768px
lg: large screen ≥992px
xl: extra large ≥1200

Example

This box has red background in small screen, blue in medium, grey in large, and green in xl
<div class="nv-text-light nv-p-4 xs:nv-bg-secondary sm:nv-bg-danger md:nv-bg-primary lg:nv-bg-light-dark xl:nv-bg-secondary">Content</div>
