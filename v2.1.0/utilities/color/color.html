Color Utilities

Apply consistent text and background colors across your project with Nova CSS color utilities.

Background Color Utilities

Class Property Color
.nv-bg-primary blue background
.nv-bg-secondary green background
.nv-bg-danger red background
.nv-bg-warning yellow background
.nv-bg-dark black background
.nv-bg-light-dark grey background
.nv-bg-light white background
.nv-bg-transparent transparent background

Example

This box has a primary background color
<div class="nv-m-4 nv-p-4 nv-bg-primary nv-text-light">Content</div>

Text Color Utilities

Class Property Color
.nv-text-primary primary text blue Text
.nv-text-secondary secondary text green text
.nv-text-danger danger text red text
.nv-text-warning warning text yellow text
.nv-text-dark dark text black text
.nv-text-light-dark light-dark text grey text

Example

This box danger background & secondary text color
<div class="nv-bg-danger nv-text-secondary nv-p-4">Content</div>
