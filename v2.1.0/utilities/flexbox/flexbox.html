Flexbox

Build dynamic layouts with Nova CSS Flexbox utilities

Note: all Nova Css Documentation are started with .nv-

Flexbox Utilities

Set element width and height easily with built-in responsive sizing helpers.

Flexbox

class Property
.nv-d-flex display: flex
.nv-d-inline-flex inline-flex
.nv-d-inline inline
.nv-d-none display: none

Example

This two box has display: flex
Which make them to stay side by side.
<div class="m-4 p-4 text-white rounded sm:nv-bg-danger md:nv-bg-primary lg:nv-bg-light-dark xl:nv-bg-secondary nv-d-flex">Content</div>

Flexbox Direction

class Property
.nv-flex-row direction: row
.nv-flex-row-reverse direction: row-reverse
.nv-flex-col direction: column
.nv-flex-col-reverse direction: col-reverse

Example

This two box has display: flex-col
Which make them to stay on top each other.
<div class="m-4 p-4 text-white rounded sm:nv-bg-danger md:nv-bg-primary lg:nv-bg-light-dark xl:nv-bg-secondary">Content</div>
