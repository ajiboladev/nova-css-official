Text

Text utilities in Nova CSS provide quick and flexible control over typography. You can easily adjust text alignment, transformation, color, and weight without writing custom CSS. These utilities help you maintain consistency and speed up your workflow when styling text elements across your project.

Note: all Nova Css Documentation are started with .nv-

Text Utilities

Control font and other typography with consistent sizing scale

Font Size Utilities

Class Property Value
.nv-fs-1 font size 2.5rem
.nv-fs-2 font size 2rem
.nv-fs-3 font size 1.75rem
.nv-fs-4 font size 1.5rem
.nv-fs-5 font size 1.25rem
.nv-fs-6 font size 1rem

Example

This text has font size 1 (.nv-fs-1)
<div class="nv-bg-primary nv-text-light nv-p-5 nv-fs-1">Content</div>

Font Weight Utilities

Text Alignment Utilities
