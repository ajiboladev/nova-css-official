Button Utilities

Create beautiful, accessible, and responsive buttons with Nova CSS button utilities. Customize appearance, states, and behavior with ease.

Note: All Nova CSS documentation classes start with .nv- prefix.

Note: In Nova CSS, Both nv-disabled and nv-btn-disabled work on any element, but only nv-btn-disabled disables buttons visually and functionally.

Button Variants & Styles
